This is a test on how to use eve with an existing mongodb database

Start the server
=================

OPen a terminal and run

::

   python eve/run.py     

View some content
=================

::

   curl -i http://localhost:5000/jobs

   
View the documentation
======================

In your browser you say

::

   curl -i http://localhost:5000/docs
